# RESEARCH UNIT TEST

## Setup Awal

Cukup jalankan :
```
npm install
```

## Run the test
1. Run this command

```
npm run start
```
OR
```
node .
```

2. Choose the one you want to run of the test or choose 'all'
3. Waiting for the test to complete
4. Check the reports of each test in testReports directory 
5. Done
